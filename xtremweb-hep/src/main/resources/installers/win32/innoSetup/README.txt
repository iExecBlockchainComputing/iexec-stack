
This directory contains Inno Setup scripts

    http://www.jrsoftware.org/isinfo.php

They are used to create installers for the win32 platform.


Available projects :
==================

   * XtremWeb-HEP server (server.iss) - fully fonctionnal
   * XtremWeb-HEP worker (worker.iss) - fully fonctionnal

How to :
======
  => first build and install as usual :
          $> make && make install

  => open Inno Setup IDE and regenerate the projects

  => copy Setup.exe file into your download page directory
     to make them available
