XtremWeb 1.5.0 for Windows

 


This page contains some informations on XtremWeb worker for Windows.

To get more complete informations about XtremWeb, please refer to the XtremWeb site.

 


Table of contents

  • What have been installed in your machine ;
  • How to uninstall the XtremWeb worker from your machine ;
  • How to get your machine desktop grid statistics.

 


What have been installed in your machine.

  • \Program Files\XtremWeb worker\xtremweb.worker.html :  this page.

  • The XtremWeb worker is installed as a Win32 service, launched everytime your machine starts.

 


How to uninstall the XtremWeb worker from your machine.

Please use the standard "add/remove application" in the configuration panel.
