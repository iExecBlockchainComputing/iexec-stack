This directory contains a Microsoft Installer project.
It is used to create an installer for the win32 platform.
Projects have been created win MSDev .NET.


Available MSI projects :
======================

   * XWHEP worker
	 
	 Oters projects are not ready yet

How to :
======
  => first build as usual :
          $> make
          $> make install

  => open MSDev .NET or any MSI project manager
     and regenerate the project

  => copy your MSI file into your download page directory
     to make it available
